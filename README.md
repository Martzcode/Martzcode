# À propos de moi

**De l'Audit à l'Ingénierie Tech : Bâtir le Futur entre Data, Code et Sécurité**

Analyser les risques, traquer les failles et comprendre la structure profonde des systèmes : voilà le fil conducteur de mon parcours. Après cinq années passées comme auditeur, j'ai choisi de basculer de l'autre côté du miroir pour concevoir et concevoir les solutions de demain.

Aujourd'hui, à la croisée des chemins entre **Software Engineering** et **Data Engineering**, je transforme des architectures complexes en solutions concrètes, fluides et performantes.
